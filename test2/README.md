# GrowBox Calculator

Этот проект представляет собой калькулятор для подбора комплектующих для гроубокса (GrowBox), разработанный для интеграции в Telegram Mini App.

## Структура проекта

- `index.html`: Главная HTML страница.
- `assets/`: Директория с ресурсами.
  - `css/style.css`: Стили приложения.
  - `js/`: Директория с JavaScript файлами.
    - `main.js`: Основной скрипт инициализации.
    - `calculator.js`: Логика калькулятора.
    - `theme.js`: Управление темой (светлая/темная).
  - `data/products.php`: Данные о товарах.
- `api/calculate.php`: PHP-скрипт для обработки вычислений (альтернатива клиентской логике).
- `README.md`: Этот файл.

## Функциональность

- Выбор количества растений и объема горшка.
- Автоматический подбор гроутента, лампы, горшков, фильтра и вентилятора.
- Отображение стоимости комплекта.
- Переключение темы интерфейса.

## Запуск

Для корректной работы проекта необходим веб-сервер с поддержкой PHP (например, Apache с mod_php или Nginx с PHP-FPM). Просто открытие `index.html` в браузере не будет работать корректно из-за ограничений на загрузку файлов с локальных путей и выполнения PHP.

1.  Установите веб-сервер (например, XAMPP, WAMP, MAMP или используйте встроенный сервер PHP `php -S localhost:8000`).
2.  Поместите все файлы проекта в корневую директорию веб-сервера.
3.  Откройте в браузере URL, соответствующий папке проекта (например, `http://localhost/growbox-calculator/`).

## Примечания

- Текущая реализация использует `fetch` для загрузки данных из `products.php`. Это позволяет централизовать данные.
- API endpoint `api/calculate.php` предоставляет альтернативный способ выполнения расчетов на сервере.
- Проект оптимизирован под мобильные устройства и работу в Telegram Mini App.
